# Cervical-Cancer-Diagnosis-DeepLearning
# Advanced Deep Learning Approach for Accurate Cervical Cancer Diagnosis

##  Project Overview
This project focuses on the classification of cervical cancer cells using advanced deep learning techniques. The system uses a hybrid deep learning approach for improving accuracy, precision, and reliability in cervical cancer diagnosis.

The project aims to assist healthcare professionals in early cancer detection through automated medical image analysis.


##  Objectives
- To develop an accurate cervical cancer classification system.
- To reduce false positives and false negatives.
- To improve medical image analysis using deep learning.
- To support early diagnosis and clinical decision-making.


##  Technologies Used
- Python
- TensorFlow
- Keras
- Flask
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn


##  Deep Learning Models Used

- Xception

### Proposed Model
- NasNetMobile

The proposed hybrid model improves classification accuracy and computational efficiency.



## ⚙️ Features

##  System Architecture
The system follows these major stages:
1. Data Collection
2. Data Preprocessing
3. Feature Extraction
4. Model Training
5. Prediction & Classification
6. Result Visualization



## Dataset
The project uses cervical cancer cell image datasets collected from publicly available medical repositories.

Preprocessing techniques used:
- Image Normalization
- Data Augmentation
- Dataset Splitting



##  Methodology
The project workflow includes:
- Image preprocessing
- Feature extraction using deep learning
- Transfer learning
- Model training and validation
- Classification using NasNetMobile
- Result evaluation using performance metrics



##  Performance Metrics
The model is evaluated using:
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC Curve

### Achieved Results
- Accuracy: 97.12%
- Precision: 96.75%
- Recall: 97.05%
- F1-Score: 96.90%
